# logger 
Server have only one option, exit on type 'e'
Client have next commands:
warn -message //send warn with text message
error -message //send error with text message
info -message //send info with text message
request_warn //request and type all warn message from server
request_error //request and type all error message from server
request_info -message //request and type all info message from server
request_since -!time1 || time1-time2 in !HH:MM:SS ||  HH:MM:SS-HH:MM:SS format //request and type all message from time1 to time2 server
exit //exit
